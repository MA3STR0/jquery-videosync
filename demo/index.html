How does it work?

Very easy. On each animated element you specify start time and class name; videsync will add this class name to your element when video playback timer matches your start-time. All effects are pure CSS.

Cool! How to install it?

Super simple. Download, add to page, set desired attributes. All instructions on Github: https://github.com/MA3STR0/jquery-videosync
